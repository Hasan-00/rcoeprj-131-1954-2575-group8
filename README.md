# IR Sensor Based Line Following Robot for Industrial Material Transport

## 📌 Abstract

This project presents the design and implementation of an IR sensor-based autonomous line-following robot, inspired by Automated Guided Vehicles (AGVs) used in industrial environments. The robot follows a predefined path using infrared sensors that detect contrast between black and white surfaces.

The system is built using an Arduino Uno microcontroller, IR sensors, a motor driver module, and DC motors mounted on a chassis. Based on sensor input, the microcontroller controls motor movement to ensure accurate navigation along the path.

This project demonstrates the concept of industrial automation where AGVs are used for efficient material transport, reducing human effort and increasing productivity.

## 🚀 Working Principle

* IR sensors detect the path (black line on white surface).
* If the robot moves off the path:

  * Sensors send signals to Arduino.
  * Arduino adjusts motor speed/direction.
* This keeps the robot aligned with the path.

---

## 🔧 Deployment / How to Run

1. Assemble the robot hardware (chassis, motors, sensors).
2. Connect IR sensors and motor driver to Arduino.
3. Upload the Arduino code using Arduino IDE.
4. Place the robot on a track (black line on white surface).
5. Power ON the robot.
6. The robot will automatically follow the path.
